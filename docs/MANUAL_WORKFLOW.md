# Manual Workflow: Pinned Comment Strategy

This guide describes how to use the Pinned Comment Strategy for YouTube videos using the manual workflow.

## Overview

We use pinned comments to drive engagement by asking viewers for their opinions and linking back to the original Reddit story.

**Process:**

1. Upload video to YouTube (Manual or via future Auto-Upload).
2. Generate comment text using the CLI.
3. Post the comment on the YouTube video.
4. Pin the comment.

## Step 1: Generate Comment Text

Use the `gossiptoon comment` command to generate the text.

### Usage

```bash
gossiptoon comment <STORY_URL> [--template <TEMPLATE_NAME>]
```

### Templates

Available templates (defined in `config/comment_templates.yaml`):

- **`engagement_default`** (Default): Standard engagement prompt. Good for most stories.
- **`engagement_controversial`**: For AITA or divisive stories. Asks "Which side are you on?".
- **`engagement_update`**: For stories with updates. Emphasizes "What happened next?".

### Examples

**Default Engagement:**

```bash
gossiptoon comment "https://reddit.com/r/AITA/..."
```

**Controversial Story:**

```bash
gossiptoon comment "https://reddit.com/r/AITA/..." --template engagement_controversial
```

**Update Story:**

```bash
gossiptoon comment "https://reddit.com/r/BORU/..." --template engagement_update
```

## Step 2: Post & Pin on YouTube

1. Go to your video on YouTube Studio or the watch page.
2. Scroll to the **Comments** section.
3. Paste the text generated by the CLI.
4. Click **Comment**.
5. Click the three dots (â‹®) on your new comment.
6. Select **Pin**.
7. Confirm by clicking **Pin** again.

## Tips

- **Timing**: Post and pin the comment immediately after the video goes public.
- **Updates**: If you upload a Part 2 video, you can update the pinned comment to link to Part 2 as well.
